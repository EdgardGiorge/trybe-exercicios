Primeira Manipulação e Criação de Imagens no Docker do Giorge

Pra criarmos um container, precisamos ter uma imagem base. Imagem é criada a partir de um documento chamado DOCKERFILE "É um arquivo de configuração onde especifica passo a passo o que deve ser executado para a criação da nossa imagem!"

    Processo de Definição
  • Criar um arquivo Dockerfile e definir uma imagem base "mesmo uma imagem em branco"
  • Definir a versão da imagem que deve ser utilizada
  • Criar a pasta onde ficará o nosso index.html, onde será iniciado o nosso site
  • Copiar o arquivo index.html pra dentro da nossa imagem
  • Definir a porta que o nosso site irá rodar
  • Setar variáveis de ambientes que podem ser utilizadas no site "Usaremos um exemplo só pra saber que é possível fazer isso"
  • Definir qual a pasta de trabalho que teremos em nossa imagem
  • Pasta que irá conter o nosso site
  • Subir o NGINX pra colocar o nosso site no ar
